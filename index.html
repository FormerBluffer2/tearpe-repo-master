What Should I Add to this Website?

I created this website as a personal learning project to learn basic coding skills. Although this website will reflect some information from my professional background, it is not my intention to use it as a website for my professional life. Instead, it will primarily be a place where I can continue to learn basic web development skills.

Since my focus is on the learning process, I intend to publish changes to the website as a work-in-progress and not worry too much about the fact that it is not as polished as a business website would expected to be. I plan to add to this website as I continue to experiment with new ideas. There will be numerous "placeholder" items that I expect to complete at a later time. Please be patient if you run into these incomplete areas. I think it is better to include placeholders rather than waiting until everything is completed. I hope this does not significantly detract from your experience.

Although this website is roughly formatted as a business website, I plan to include various random elements that I have created to expand my coding knowledge. These will generally be included in an area I am calling My Experimental Projects. For example, I have used freeCodeCamp as a resource and they have mini projects that can be completed to learn new skills. I plan to include some of these projects.

I welcome any constructive feedback that you care to provide. Also, I would greatly appreciate any suggestions for new content that I can add.

Thank you for taking the time to visit my site.
